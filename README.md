# 🧪 Selenium TestNG Automation Project

This project is a basic automation testing framework built using **Selenium WebDriver** and **TestNG** with **Java** and **Maven**.

---

## 🚀 Features

- Automated User Registration Test
- Assertion-based validation
- Dynamic email generation
- TestNG annotations (@Test, @BeforeMethod, @AfterMethod)
- Maven project structure

---

## 🛠️ Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven
- ChromeDriver
